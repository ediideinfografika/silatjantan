Very small module (pluging) that add a geojson layer for the geolocation leaflet module.


Useful:
------
  http://geojson.io/
  https://leafletjs.com/examples/geojson/
  https://leafletjs.com/reference-1.7.1.html#path-stroke
  https://fr.wikipedia.org/wiki/GeoJSON
  https://geojson.org/
  https://france-geojson.gregoiredavid.fr/


Warning:
----------
For the style to be taken into account, the following properties must be used:

- "stroke" became "color"
- "stroke-width" became "weight"
- "stroke-opacity" became "opacity"
- "fill" became "fillColor"
- "fill-opacity" became "fillOpacity"
